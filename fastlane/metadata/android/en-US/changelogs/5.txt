Housekeeping release ahead of F-Droid submission. The app's package name changed from the placeholder com.example.videowallpaper to io.github.codeg0blin.videowallpaper, since F-Droid requires a unique identifier tied to a domain the developer controls. No functional changes. Because the package name changed, this is treated as a new app by Android — if you have a previous version installed, you'll need to uninstall it first and install this one fresh; it will not update in place.
